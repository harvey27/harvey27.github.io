## Big O Notation

{% highlight javascript %}
use admin
db.createUser{
	user: "bonitao",
	pwd: "2016bonitao",
	roles: [{role: "userAdminAnyDatabase", db: "admin"}]
}
{% endhighlight %}

<strong> O(1) </strong>

<strong> O(log n) </strong>

<strong> O(n) </strong>

<strong> O(n^2) </strong>

<strong> O(n log n) </strong>
